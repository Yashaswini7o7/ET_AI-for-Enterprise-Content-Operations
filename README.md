# AI Content Ops (Offline Version)

## Features
- Content generation (mock AI)
- Compliance checking
- Translation (mock)
- Streamlit UI

---

## Run Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

---

## Run Frontend
cd frontend
pip install -r requirements.txt
streamlit run app.py

---

## URLs
Backend: http://localhost:8000
Frontend: http://localhost:8501
